# Faces of the World

Anonymous face wall. Post your face from anywhere on Earth.

A spin-off from [ArtOut](https://artout.freeappstore.online) — people kept posting selfies instead of street art, so we gave them their own app.

## Stack

- React 19 + Vite + TypeScript + Tailwind CSS 4
- FAS SDK (auth, Collections DB)
- Cloudinary (face-centered image transforms)
- Nominatim (reverse geocoding for city + country)

## Dev

```bash
pnpm install
pnpm dev
```
